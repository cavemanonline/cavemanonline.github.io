第28課　いろいろな経験をさせてもらって、楽しかったです

１．Ｖ（さ）せる　＜使动＞

动词后接「（さ）せる」构成动词的使动态，表达让某人做某事的含义。
「Ｖ（さ）せる」还可以按照Ⅱ类动词的活用规则进行变形。
动词使动态的变形方式：
Ⅰ类动词（五段动词）：
　　规则：Ｖ[词尾う段假名]　→　Ｖ[词尾あ段假名]＋せる
　　示例：言う　→　言わせる
　　　　　書く　→　書かせる
　　　　　消す　→　消させる
　　　　　持つ　→　持たせる
　　　　　死ぬ　→　死なせる
　　　　　運ぶ　→　運ばせる
　　　　　飲む　→　飲ませる
　　　　　走る　→　走らせる

Ⅱ类动词（一段动词）：
　　规则：Ｖ[词尾る]　→　Ｖ[去除词尾る]＋させる
　　示例：食べる　→　食べさせる
　　　　　寝る　→　寝させる

Ⅲ类动词（カ变、サ变动词）：
　　规则：来る　→　来させる（こさせる）
　　　　　する　→　させる
以动词的使动态做谓语的句子称为使动句。使动句与主动句的不同在于出现了以「は/が」表示的使动者。

A. 他动词使动句

他动词使动句中，使动者做主语，用「は/が」表示；动作主体用「に」表示，动作客体用「を」表示。
主动句：子どもが 野菜を 食べる。
被动句：お母さんは 子どもに 野菜を 食べさせる。

B. 自动词使动句

自动词使动句中，使动者做主语，用「は/が」表示；动作主体用「を/に」表示。
主动句：子どもが 買い物に 行く。
被动句：お母さんは 子どもを/に 買い物に 行かせる。

２．Ｖ（さ）せられる　＜使动被动＞

使动句和被动句合成的句子称为「使动被动句」，表示动作主体在他人的强制、命令、要求下被迫、不情愿地做某事。相当于汉语的「被迫……；不得不……；无奈……」。
「Ｖ（さ）せられる」还可以按照Ⅱ类动词的活用规则进行变形。
Ⅰ类动词的使动被动形式「Ｖ[あ段假名]＋せられる」，常约音为「Ｖ[あ段假名]＋される」，这种形式也更为常见。
动词使动被动态的变形方式：
Ⅰ类动词（五段动词）：
　　规则：Ｖ[词尾う段假名]　→　Ｖ[词尾あ段假名]＋される（せらせる）
　　示例：言う　→　言わされる
　　　　　書く　→　書かされる
　　　　　消す　→　消さされる
　　　　　持つ　→　持たされる
　　　　　死ぬ　→　死なされる
　　　　　運ぶ　→　運ばされる
　　　　　飲む　→　飲まさせる
　　　　　走る　→　走らされる

Ⅱ类动词（一段动词）：
　　规则：Ｖ[词尾る]　→　Ｖ[去除词尾る]＋させられる
　　示例：食べる　→　食べさせられる
　　　　　見る　→　見させられる

Ⅲ类动词（カ变、サ变动词）：
　　规则：来る　→　来させられる（こさせられる）
　　　　　する　→　させられる
使动句：お母さんは　子どもに　野菜を　食べさせる。
使动被动句：子どもは　お母さんに　野菜を　食べさせられる。
使动被动句中，动作主体用「は／が」表示，使动者用「に」表示，但常常不出现使动者。
