１．Ｖ（ら）れる　＜被动＞

动词被动态由动词后接「（ら）れる」构成，表达被动的含义。「Ｖ（ら）れる」还可以按照Ⅱ类动词的活用规则进行活用。
动词被动态（也称受身形）的变形方式：
Ⅰ类动词（五段动词）：
　　规则：Ｖ[词尾う段假名]　→　Ｖ[词尾あ段假名]＋れる
　　示例：言う　→　言われる
　　　　　書く　→　書かれる
　　　　　写す　→　写される
　　　　　持つ　→　持たれる
　　　　　死ぬ　→　死なれる
　　　　　運ぶ　→　運ばれる
　　　　　噛む　→　噛まれる
　　　　　触る　→　触られる

Ⅱ类动词（一段动词）：
　　规则：Ｖ[词尾る]　→　Ｖ[去除词尾る]＋られる
　　示例：食べる　→　食べられる
　　　　　見る　→　見られる

Ⅲ类动词（カ变、サ变动词）：
　　规则：来る　→　来られる（こられる）
　　　　　する　→　される
以动词的被动态作谓语的句子称为被动句，被动句是表达句中主语所示的人或物受到某种作用或影响的句子。
日语的被动句分为直接被动句、间接被动句和物主被动句。

A. 直接被动句

主动句：先生が　太郎を　叱った。
被动句：太郎が　先生に　叱られた。
主动句：先生が　学生に　質問をした。
被动句：学生が　先生に　質問をされた。
例句1～10为直接被动句的例子。
在例句1～6中，主语均为指人名词（有时可省略）。
除此之外，如例句7～10所示，事物名词也可以充当被动句的主语。这样的被动句多用于客观叙述某一现象，动作的主体往往不明或非特定，所以在句中经常不出现。

（１）上田さんはクラスメートに笑われてしまった。（上田被同学笑话了。）

（２）先生に褒められてうれしかった。（被老师表扬了所以很开心。）

（３）李さんは佐藤さんに食事に招待された。(小李被佐藤邀请去吃饭了。)

（４）同僚に誘われて、ピクニックに行った。（受同事邀请，去郊游了。）

（５）張さんは森山先生にコピーを頼まれた。（小张被森山老师拜托复印。）

（６）町を歩いたら、外国人に道を聞かれた。（走在街上，就被外国人问了路。）

（７）この本は当時の若者に広く読まれていた。（这本书被当时的年轻人广泛阅读。）

（８）日本語は英語より難しいと言われている。（据说日语比英语更难。）

（９）日本では入学式は４月に行なわれる。（在日本，入学典礼在4月举行。）

（10）2008年のオリンピックは北京で開かれた。（2008年的奥运会在北京举办。）

B. 间接被动句

主动句：帰りに　雨が　降った。
被动句：私は　帰りに　雨に　降られた。
间接被动句表示某一事态的发生间接地对主语所指的人造成了不良的影响或损害。用「は／が」表示的主语是原主动句中无法出现的名词，原主动句中的动作主体在被动句中变为补足语，用「に」表示。与其他类型的被动句不同，这类被动句的谓语动词可以是自动词。

（１）電車で香水の強い人に隣に座られて、苦しかった。（乘坐电车，被香水很强的人坐在旁边，很难受。）

（２）こんなところで寝られたら困るよ。（在这种地方睡觉很为难呀。）

（３）遅刻してしまって先生に怒られた。（迟到了，被老师骂了。）

（４）あの社長は有能な社員に辞められて困っている。（那个社长正被有能力的公司职员辞职所困扰。）

C. 物主被动句

主动句：男の人が　私の　足を　踏んだ。
被动句：私は　男の人に　足を　踏まれた。
物主被动句表示物主遭受了不良影响或损失。主语是原主动句中动作客体的所有者（物主），用「は／が」表示，原主动句中动作的客体不变，仍用「を」表示，原主动句中的动作主体变为补足语，用「に」表示。

（１）弟は兄にリンゴを食べられて泣き出した。（弟弟被哥哥吃了苹果而哭起来了。）

（２）昨日友達にパソコンを壊されてしまいました。（昨天被朋友弄坏了电脑。）

（３）電車で財布を盗まれたのに気付かなかった。（在电车上钱包被偷了却没注意。）

（４）子どものころ、親に日記を読まれて非常に不愉快だった。（孩童时期，被父母看日记非常不愉快。）

２．Ｖてばかりいる　＜动作的持续＞

助词「ばかり」表示限定，相当于「只……；净……」，可以直接接在名词后使用，也可构成一些特定的句式。
「Ｖてばかりいる」表示某一状态一直持续或某一动作不断反复，含有说话人不满、批评、责备的语气。相当于汉语的「光……；老是……」。

（１）あの子は遊んでばかりいて、全然勉強しない。

（２）パソコンを見てばかりいると、目が疲れてしまう。

（３）あのころは何もしないで、毎日寝てばかりいた。

（４）最近は上司に怒られてばかりいる。

３．Ｖたばかりだ　＜动作刚刚结束＞

动词「Ｖた」的形式后接「ばかり」，表示Ｖ这一动作刚刚发生或完成，相当于「刚……；刚……不久」。
后接名词时，用「ＶたばかりのＮ」的形式。

（１）日本語を習い始めたばかりだから、日本語の新聞はまだ読めません。

（２）昼ご飯を食べたばかりだから、何も食べたくないです。

（３）北京に来たばかりで、まだこちらの生活に慣れていない。

（４）買ったばかりの車を盗まれてしまった。

（４）授業が始まる

４．Ｖことはない　＜建议＞

「ことはない」接在动词词典形后面，表示没有必要做某事。相当于汉语的「不必……；不用……」。

（１）そんなことで悩むことはないよ。

（２）まだ時間があるから、急ぐことはない。

（３）他人の目や評価を気にすることはない。

（４）日本語が分からなくても、中国語の説明があるので心配することはない。

基练题８. Ｖことはない

例　劉さんは冗談で言っている・気にする
　→　劉さんは冗談で言っているから、気にすることはないよ。

（１）おもちゃだ・怖がる

→　おもちゃだから、怖がることはないよ。

（２）時間はまだ大丈夫だ・慌てる

→　時間はまだ大丈夫だから、慌てることがないよ。

（３）大したことはない・心配する

→　大したことはないから、心配することがないよ。

（４）子どもの言うことだ・そんなに怒る

→　子どもの言うことだから、そんなに怒ることがないよ。

５．Ｖことがある　＜频率低＞

「ことがある」接在动词的词典形或否定形后面，表示有时或偶尔会发生某事，相当于汉语的「有时……；偶尔……」。要注意这个句式与表示经历的「Ｖたことがある」形式上有区别。

（１）朝寝坊して授業に遅れることがあります。

（２）仕事が忙しい時は、徹夜することがある。

（３）台風などの影響で新幹線が止まることがある。

（４）メートを送っても届かないことがあります。

基练题９. Ｖことがある

例　母はときどき怒る
　→　母はときどき怒ることがあります。

（１）仕事で徹夜する

→　仕事で徹夜することがあります。

（２）ときどきこの道を通る

→　ときどきこの道を通ることがあります。

（３）電車の中で足を踏まれる

→　電車の中で足を踏まれることがあります。

（４）上司に褒められる

→　上司に褒められることがあります。

６．Ｖところだ／Ｖているところだ／Ｖたところだ　＜动作的开始、进行和结束＞

「ところ」在这里表示动作所处的时间阶段。

口语中「ところだ」的「だ」可以省略。

A. Ｖところだ

「Ｖところだ」表示说话时动作正要开始进行。

（１）これから出かけるところです。

（２）もうすぐ会議が始まるところだ。

（３）夕食はカレーにしようと思って材料を買いに行くところです。

B. Ｖているところだ

「Ｖているところだ」表示说话时动作正在进行之中。

（１）今、会議の準備をしているところです。

（２）今、餃子を作っているところだけど、食べに来ない？

（３）A：もしもし、今、何しているの？

　　　B：今宿題をやっているところ。

C. Ｖたところだ

「Ｖたところだ」表示说话时动作刚刚结束。

（１）今、旅行から帰ってきたところです。

（２）今、コンサートが終わって会場を出たところです。

（３）A：遅れてすみません。

　　　B：いえ、私も今来たところです。

７．Ｖ（よ）とする　＜意图＞

「とする」接在自主动词的意志形后面，表示试图完成该动作。

经常以「Ｖ（よ）うとしたところ」「Ｖ（よ）うとした時」的形式，表示正要进行该动作的时间点。相当于汉语的「正要……的时候」。

（１）王さんは日本語能力試験の２級を受けようとしている。

（２）彼に本当のことを話そうとしたが、なかなか言えなかった。

（３）そのことを忘れようとしても忘れられない。

（４）今、電話しようとしたところだ。

（５）釣りに行こうとした時、雨が降り出した。

基练题13. Ｖ（よ）とする

例　ドアを開ける・だめだった　→　ドアを開けようとしたが、だめだった。

（１）出かける・電話が鳴った

→　出かけようとしたが、電話が鳴った。

（２）彼から離れる・離れられない

→　彼から離れようとしたが、離れられない。

（３）電車に乗る・駅員に呼び止められた

→　電車に登ろうとしたが、駅員に呼び止められた。

（４）彼の提案を受け入れる・反対の声が強くてできなかった

→　彼の提案を受け入れようとしたが、反対の声が強くてできなかった。

８．Ｎの／Ｖたびに　＜反复＞

「たびに」接在「名词＋の」（名词一般为动作性名词）或动词词典形后面，表示每次进行该动作或发生该变化时，都发生同样的情况。相当于汉语的「每当……；每次……」。

（１）北京出張のたびに、そのホテルに泊まる。

（２）アメリカへ行くたびにたくさん買い物をして来る。

（３）この歌を聞くたびに、涙が止まらない。

（４）嫌なことがあるたびにお酒を飲んでしまう。

基练题14. Ｎのたびに

例　北京出張・このホテルに泊まる
　→　北京出張のたびにこのホテルに泊まります。

（１）会議・ノートを取る

→　会議のたびにノートを取リます。

（２）デート・送り迎えをしてくれる

→　デートのたびに送り迎えをしてくれます。

（３）雨・別れた彼のことを思い出す

→　雨のたびに別れた彼のことを思い出します。

（４）買い物・ポイント・たまる

→　買い物のたびにポイントがたまります。

基练题15. Ｖたびに

例　この映画を見る・涙が出る　→　この映画を見るたびに涙が出ます。

（１）あの２人は会う・けんかする

→　あの２人は会うたびにけんかします。

（２）雨が降る・足が痛くなる

→　雨が降るたびに、足が痛くなります。

（３）この島を訪れる・自然の美しさに感動する。

→　この島を訪れるたびに、自然の美しさに感動します。

（４）李さんは来る・お土産を持ってきてくれる

→　李さんは来るたびに、お土産を持ってきてくれます。

９．くらい　＜最低限度＞

「くらい」接在名词或动词、形容词的连体形后面，表示最低的限度。也作「ぐらい」。

（１）そんなことくらいできるでしょう。

（２）忙しくても食事をするぐらいの時間はあるでしょう。

（３）風邪を引いたぐらいで病院に行くことはない。

（４）外国に行くなら、挨拶くらいはできるようにしたほうがいい。

10．口语中的缩略形式

在日常口语中，有些形式使用其缩略形式表达。
では　→ じゃ
〜ている　→ 〜てる
〜ておく　→ 〜とく
〜ては　→ 〜ちゃ
〜てしまう　→ 〜ちゃう

（１）じゃ、飲み物を用意しとくね。

（２）あ、早く行かなくちゃ。

（３）そこで待ってて。入っちゃだめよ。

（４）ごめん。忘れちゃった。

基练题17. ちゃ

例　荷物を無くした
　→　荷物を無くしてしまった。
　→　荷物を無くしちゃった。

→　雨に降られちゃった。

（２）お皿を割った

→　お皿を割ってしまった。

→　お皿を割っちゃった。

（３）学校に遅刻した

→　学校に遅刻してしまった。

→　学校に遅刻しちゃった。

（４）話してはいけないことを話した

→　話してはいけないことを話してしまった。

→　話してはいけないことを話しちゃった。

→　雪が降り出しました。

→　雪が降り続けています。

（３）選手たち・走る

→　選手たちが走り始めました。

→　選手たちが走り出しました。

→　選手たちが走り続けています。

（４）子ども・泣く

→　子どもが泣き始めました。

→　子どもが泣き出しました。

→　子どもが泣き続けています。

  1. 打つ：うつ。 ↩︎
  2. 諦める（あきらめる）④：断念，死心；放弃想法、念头。 ↩︎
  3. 出す：［补助动词］……起来。 ↩︎
