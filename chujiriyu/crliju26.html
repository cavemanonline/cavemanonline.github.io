初级日语例句和基练笔记总目录

第26課　電源をオフのままにしてしまいました

１．Ｖてしまう　＜不良后果＞

本课中「しまう」接在动词的「て形」后，此动作一般会导致不良的后果。此句表达了说话人遗憾、后悔的语气，多用「Ｖてしまった」的形式。

（１）朝寝坊したから、授業に遅刻してしまいました。

（２）傘を地下鉄に忘れてしまいました。

（３）パソコンが壊れてしまったので、午後修理に行く予定だ。

（４）風邪を引いてしまうよ。コートを着て行ったほうがいい。

基练题１. Ｖてしまう

例　会社に遅れる　→　会社に遅れてしまいました。

（１）風邪を引く1

→　風邪を引いてしまいました／てしまった。

（２）財布を落とす2

→　財布を落としてしまいました。

（３）パソコンを壊す3

→　パソコンを壊してしまいました。

（４）宿題を忘れる4

→　宿題を忘れてしまいました。

２．Ｎの／Ｖた／Ｖないまま　＜保持原状＞

「まま」是名词，前接「名词＋の」以及动词的过去时或否定式，表示「保持着原样；任凭某种状态持续」的意思。

（１）靴のままで部屋に入ってはいけません。

（２）きゅうりは生のまま食べたほうがおいしいです。

（３）夕べ5、エアコンをつけたまま寝てしまったから、風邪を引いた。

（４）急いでいたので、電気も消さないまま出かけて行った。

基练题２. Ｎのままだ

例　生・食べる　→　生のまま食べました。

（１）靴・部屋に入る

→　靴のままで部屋に入りました。

（２）パジャマ・外へ出る

→　パジャマのままで外へ出ました。

（３）水着・温泉に入る

→　水着のままで温泉に入りました。

（４）皮・食べる

→　皮のまま食べました。

基练题３. Ｖた／ないままだ

例　靴を履く・部屋に入る　→　靴を履いたまま部屋に入らないでください。

（１）クーラーをつける・寝る

→　クーラーをつけたまま寝ないでください。

（２）電気をつける・家を出る

→　電気をつけたまま家を出ないでください。

（３）告白しない・卒業する

→　告白しないまま卒業しないでください。

（４）テレビを消さない6・寝る

→　テレビを消さないまま寝ないでください。

３．Ｎ１をＮ２にする　＜使然＞

表示将Ｎ1变成Ｎ2，或使Ｎ1成为Ｎ2所示的状态。

（１）この部屋を資料室にしましょう。

（２）教科書に書いてあるものを自分の知識にしなければならない。

（３）うっかりして、電源をオフのままにしてしまった。

（４）このボタンを押すと、携帯をかけられない状態にすることができる。

基练题４. Ｎ１をＮ２にする

例　この部屋・子ども部屋　→　この部屋を子ども部屋にしましょう。

（１）スイッチ・オフ

→　スイッチをオフにしましょう。

（２）ここ・会議室

→　ここを会議室にしましょう。

（３）この映画・教材

→　この映画を教材にしましょう。

（４）これ・みんなの仕事

→　これをみんなの仕事にしましょう。

４．ため（に）　＜原因＞

「ために」接在动词的简体形式或「名词＋の」的后面，表示是后句所述内容的原因。相当于汉语的「因为……；由于……」等。
「ために」的「に」可省略。

（１）病気のために、今日は一日休みました。

（２）雪のため、新幹線が止まりました。

（３）友達が突然訪ねてきたために、約束の時間に遅れてしまった。

（４）昨日徹夜したため、あまり元気がないです。

基练题５. Ｎのため（に）

例　風邪・会社を休んだ　→　風邪のために会社を休みました。

（１）渋滞7・遅刻する

→　渋滞のために遅刻します。

（２）不注意・転ぶ

→　不注意のために転びます。

（３）台風・電車が遅れる

→　台風のために電車が遅れます。

（４）仕事・ストレスがたまっている

→　仕事のためにストレスがたまっています。

基练题６. Ｖため（に）

例　風邪を引いた・会社を休んだ
　→　風邪を引いたために会社を休みました。

（１）パソコンを買った・お金がなくなった

→　パソコンを買ったためにお金がなくなりました。

（２）花粉症の薬を飲んでいる・眠くなった

→　花粉症の薬を飲んでいるために眠くなりました。

（３）友達が来た・勉強できなかった。

→　友達が来たために勉強できませんでした。

（４）パソコンを持っていない・メールが使用できない

→　パソコンを持っていないためにメールが使用できません。

５．Ｖようだ　＜推测＞

「ようだ」接在「名词＋の」或用词连体形后面，表示说话人根据自己的观察、感觉、印象等作出的主观推测。相当于汉语的「好像……；大概……」。

（１）雨がやんだようだよ。

（２）今年は就職が厳しいようです。

（３）あの人は李さんの妹さんのようです。

（４）簡単なようだが、実際にやってみるとなかなかうまくいかない。

基练题７. Ｖようだ

例　雨が降っている　→　雨が降っているようだ。

（１）張さんは試験に合格した

→　張さんは試験に合格したようだ。

（２）風邪をひいた

→　風邪をひいたようだ。

（３）李さんはもうすべてを知っている

→　李さんはもうすべてを知っているようだ。

（４）このパソコンはまだ使える

→　このパソコンはまだ使える8ようだ。

６．Ｎのような／ように；Ｎみたいな／みたいに　＜示例＞

「Ｎのようだ／Ｎみたいだ」除了此前学过的表示比喻的用法外，还可以用于举例。意为「像……那样的……」。
用于举例时，通常不用「～だ」的形式，而采用「～な」的形式接名词或「～に」的形式接动词。

（１）北京や上海のような大都市は人が多いです。

（２）李さんみたいなやさしい人になりたいですね。

（３）しっかり勉強しなさい。今のように、毎日ごろごろしていてはいけないよ。

（４）日本語能力試験を受けるなら、張さんみたいに今から準備しておいたほうがいいでしょう。

基练题８. Ｎのような

例　北京・大都市に就職したい
　→　北京のような大都市に就職したい。

（１）春・天気が続いている

→　春のような天気が続いている。

（２）お城9・家に住みたい

→　お城のような家に住みたい。

（３）男の子・かっこう10をしている

→　男の子のようなかっこうをしている。

（４）８月11・暑さ12だ

→　８月のような暑さだ。

基练题９. Ｎのように

例　この町は夏になるとにぎやかだ・大都会
　→　この町は夏になると大都会のようににぎやかです。

（１）弟はテニスが上手だ・プロの選手13

→　弟はプロの選手のようにテニスが上手です。

（２）先生はやさしい・母

→　先生は母のようにやさしいです。

（３）飛びたい・鳥

→　鳥のように飛びたいです。

（４）彼女は笑った・子ども

→　彼女は子どものように笑いました。

基练题10. Ｎみたいな

例　わたしの故郷は古い町です・西安
　→　わたしの故郷は西安みたいな古い町です。

注释：古い14

（１）今日はいい一日でした・夢

→　今日は夢みたいないい一日でした。

（２）おかしい考え方です・子ども

→　子どもみたいなおかしい考え方です。

（３）寒さが続いている・冬

→　冬みたいな寒さが続いている。

（４）家には広い庭がある・公園

→　家には公園みたいな広い庭がある。

基练题11. Ｎみたいに

例　張さん・頑張る　→　張さんみたいに頑張ります。

（１）父・立派になりたい

→　父みたいに立派になりたいです。

（２）李さん・うまく写真を撮りたい

→　李さんみたいにうまく写真を撮りたいです。

（３）12月・寒い

→　12月みたいに寒いです。

（４）高校・大学も宿題が多い

→　高校みたいに大学も宿題が多いです。

７．Ｖず（に）　＜否定的状态＞

「Ｖずに」是「Ｖないで」的书面语形式，动词接「ず」与接「ない」的形式形同，例外是「する」要变成「せず」。「に」可以省略。
「Ｖずに」的意义用法与「Ｖないで」基本相同，可以表示「不做前项动作的情况下做后项动作」，如例句1～3；也可以表示「不做（不是）前项，而做（而是）后项」，如例句4、5。

（１）朝ご飯も食べずに学校に行きました。

（２）辞書を引かずに日本語の新聞を読めるようになりました。

（３）彼は１日も休まずに、まじめに働いています。

（４）週末はどこへも行かずに寮で勉強をするつもりです。

（５）お金はすぐにはもらえず、来月まで待たなければならない。

基练题12. Ｖず（に）

例　傘をさす・歩く　→　傘をささずに歩きます。

（１）休む・働いている

→　休まずに働いています。

（２）塩を使う・料理を作った

→　塩を使わずに料理を作りました。

（３）諦める15・頑張ろう

→　諦めずに頑張りましょう。

（４）窓を閉める・出かけた

→　窓を閉めずに出かけました。

（５）カードを持つ・買い物に出かけた

→　カードを持たずに買い物に出かけました。

（６）確認する・送ってしまった

→　確認せずに送ってしまいました。

８．Ａ／Ｖすぎる　＜程度＞

「～すぎる」接在动词接「-ます」的形式和形容词词干后面，用于表示动作、行为或状态超过了一般的程度或范围。相当于汉语的「过于……；太……」等。

（１）字が小さすぎて、はっきり見えません。

（２）高すぎて、とても買えません。

（３）わたしの家の周りは静かすぎて、ちょっと寂しいです。

（４）お酒を飲みすぎて、気持ち悪いです。

基练题13. Ａすぎる

例　この荷物・重い　→　この荷物は重すぎます。

（１）このケーキ・甘い

→　このケーキは甘すぎます。

（２）あの子・細い

→　あの子は細すぎます。

（３）この布団・かたい

→　この布団はかたすぎます。

（４）この部屋・狭い

→　この部屋は狭すぎます。

基练题14. Ｖすぎる

例　久しぶりに友達と会う・お酒を飲んだ
　→　久しぶりに友達と会って、お酒を飲みすぎました。

（１）料理がおいしい・食べた

→　料理がおいしくて、食べすぎました。

（２）お土産をたくさん買う・お金を使った

→　お土産をたくさん買って、お金を使いすぎました。

（３）今週は仕事が忙しい・働いた

→　今週は仕事が忙しくて、働きすぎました。

（４）あの人は口が軽い・何でもしゃべる

→　あの人は口が軽くて、何でもしゃべりすぎます。

９．お–／ご–　＜表示尊敬的前缀＞

「お」「ご」用于名词前面，是表示尊敬的前缀。
「ご」一般后接汉语词（音读词），「お」后接和语词（训读词）。但也有例外，如：「お電話、お返事、ご返事。」

（１）失礼ですが、お名前は？

（２）心からお詫びします。

（３）何かご意見はありませんか。

（４）ご連絡ありがとうございました。

基练题15. お／ごＮ

例　礼　→　お礼

（１）花

→　お花

（２）弁当16

→　お弁当

→　ご返事／お返事

→　お電話

（５）案内

→　ご案内

（６）都合

→　ご都合

（７）説明

→　ご説明

（８）相談

→　ご相談
